# Pulse 台股決策台

把台股價格、全球新聞、供應鏈事件與多模型預測放在同一個每日研究畫面。

## 目前功能

- 台股市場溫度、指數與匯率卡片
- 76 檔截圖標的（個股與 ETF）的 3 日、1 週、2 週、1／2／3 個月預測
- 趨勢、動能、均值回歸、集成中位數四種模型可同時顯示或個別關閉
- 逐段向前回測，顯示方向命中率、平均誤差與可靠度
- 依自選群組、產業、供應鏈、上下游階段與關係標籤搜尋、篩選
- 最多同時選取 4 檔建立比較清單，並顯示同產業／同供應鏈關聯股
- 世界新聞、台灣新聞與供應鏈事件篩選
- 事件 → 供應鏈環節 → 股票的影響路徑
- TWSE／TPEx 公開歷史資料連線失敗時，自動退回並明確標示「示範快照」
- 手機、平板與桌面版響應式介面

## 啟動

```bash
chmod +x ./start-app.sh
./start-app.sh 8082
```

本專案固定使用 port `8082`，瀏覽器開啟 `http://127.0.0.1:8082`。也可直接執行：

```bash
python3 market_server.py --port 8082
```

## 資料與模型說明

- 上市股票與 ETF 歷史價格：臺灣證券交易所「個股日成交資訊」公開報表
- 上櫃股票歷史價格：證券櫃檯買賣中心「個股日成交資訊」公開報表
- 興櫃／創新板新掛牌標的若尚無足夠歷史資料，會保留可辨識的示範快照
- 台股加權指數：臺灣證券交易所「市場成交資訊」公開報表
- 新聞：公開 Google News RSS，標題經規則分類成台灣／全球／供應鏈
- 模型：趨勢迴歸、短中期動能、均值回歸，以及三者的集成中位數
- 驗證：用歷史資料逐段 walk-forward，依選定期間重新計算方向命中率與平均誤差

這是一套研究工具，不是自動交易系統。預測是機率與情境範圍，不構成投資建議或報酬保證。
