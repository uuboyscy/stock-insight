# Pulse 台股決策台

把台股價格、全球新聞、供應鏈事件與多模型預測放在同一個每日研究畫面。

## 目前功能

- 台股市場溫度、指數與匯率卡片
- 台積電、鴻海、聯發科、廣達的 3 日、1 週、2 週、1／2／3 個月預測
- 趨勢、動能、均值回歸、集成中位數四種模型可同時顯示或個別關閉
- 逐段向前回測，顯示方向命中率、平均誤差與可靠度
- 世界新聞、台灣新聞與供應鏈事件篩選
- 事件 → 供應鏈環節 → 股票的影響路徑
- TWSE 公開歷史資料連線失敗時，自動退回並明確標示「示範快照」
- 手機、平板與桌面版響應式介面

## 啟動

```bash
chmod +x ./start-app.sh
./start-app.sh 8080
```

瀏覽器開啟 `http://127.0.0.1:8080`。也可直接執行：

```bash
python3 market_server.py --port 8080
```

## 資料與模型說明

- 上市股票歷史價格：臺灣證券交易所「個股日成交資訊」公開報表
- 台股加權指數：臺灣證券交易所「市場成交資訊」公開報表
- 新聞：公開 Google News RSS，標題經規則分類成台灣／全球／供應鏈
- 模型：趨勢迴歸、短中期動能、均值回歸，以及三者的集成中位數
- 驗證：用歷史資料逐段 walk-forward，依選定期間重新計算方向命中率與平均誤差

這是一套研究工具，不是自動交易系統。預測是機率與情境範圍，不構成投資建議或報酬保證。
